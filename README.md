# 🚀 SpaceX Falcon 9 Launch Success Prediction

## 📌 Project Overview
This is an end-to-end Data Science project based on SpaceX Falcon 9 launch data.  
The project focuses on collecting, processing, analyzing, and predicting rocket launch success using Machine Learning techniques.

The project includes:
- API Data Collection
- Web Scraping
- Data Wrangling
- SQL Analysis
- Exploratory Data Analysis (EDA)
- Interactive Visualizations
- Machine Learning Models
- Interactive Dashboard using Dash & Plotly

# 📊 Project Workflow

## 1️⃣ Data Collection
- Collected SpaceX launch data using API
- Extracted additional data using web scraping

## 2️⃣ Data Wrangling
- Cleaned missing values
- Handled categorical data
- Prepared dataset for analysis

## 3️⃣ SQL Analysis
- Performed SQL queries using SQLite
- Extracted useful business insights

## 4️⃣ Exploratory Data Analysis (EDA)
- Visualized launch trends
- Analyzed payload impact
- Compared launch site performance

## 5️⃣ Machine Learning
Built classification models to predict launch success:
- Logistic Regression
- SVM
- Decision Tree
- KNN

## 6️⃣ Interactive Dashboard
Created an interactive dashboard using Dash and Plotly:
- Launch site selection
- Payload filtering
- Success rate visualization
- Scatter plots
